# Net Life project

This project is multiplayer version of [Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)

## Build
`$ go build .`

## Usage
### SinglePlayer mode
`$ ./net-life`
### MultiPlayer mode
Run on the server-machine:
`$ ./net-life server`
Run on the one of client-machines:
`$ ./net-life mp`
Run on the another client machine:
`$ ./net-life mp`
